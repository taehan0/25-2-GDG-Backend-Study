## Repository
- 데이터베이스에 접근하여 CRUD 작업을 수행하는 계층
- Create: 데이터 생성
- Read: 데이터 조회
- Update: 기존 데이터 수정
- Delete: 데이터 삭제

## ORM
- 객체와 관계형 데이터베이스와의 데이터를 자동으로 매핑해주는 것
## JPA
- 자바 진영의 ORM 기술 표준
- 자바에서 객체를 데이터베이스에 저장하고 관리하기 위한 인터페이스와 기능을 제공하는 API

## JPA 동작방식
- 엔티티 매니저를 통해 엔티티 영속성 컨텍스트 공간에서 관리
- 변경 내용을 즉시 DB에 반영하지 않고 트랙잭션이 커밋될 때 필요한 SQL을 한번에 실행
- 엔티티의 변경을 자동으로 감지해 SQL을 생성, 데이터베이스와 동기화를 수행
- 이미 조회된 엔티티는 영속성 컨텍스트에 저장되어 동일 트랜잭션 내에서 재사용됨
